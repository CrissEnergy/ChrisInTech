# **App Name**: Chris In Tech Portfolio

## Core Features:

- Navigation Bar: Sticky navigation bar with smooth scrolling to different sections.
- Hero Section: Eye-catching hero section to introduce Chris In Tech and his skills.
- Skills Showcase: Display skills in a categorized grid layout.
- Project Showcase: Showcase projects with images, descriptions, and links to live demos and GitHub repositories.
- Contact Form: A functional contact form for visitors to send messages directly.
- Social Media Links: Link to professional social media profiles.
- Animation on scroll: Subtle animations on scroll as the user is exploring the webpage

## Style Guidelines:

- Primary color: Deep blue (#0A192F) for a professional and modern look.
- Background color: Slightly desaturated deep blue (#162438).
- Accent color: Vibrant green (#64FFDA) for highlights and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, machined feel; readable in headings or body text. 
- Use Font Awesome icons for skills and social media links.
- Use Flexbox and Grid for a responsive layout that adapts to different screen sizes.
- Subtle CSS animations and transitions to enhance user experience.
